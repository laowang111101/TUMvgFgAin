# 简易图书馆管理系统 python326

## 项目概述

本系统是基于Python语言的图书馆管理系统，通过Tkinter构建用户操作界面，并使用MySQL数据库进行数据存储。系统集成了bcrypt密码加密、正则表单验证等安全功能，提供了图书管理全流程的支持。

## 技术栈

- **后端语言**：Python
- **前端框架**：Tkinter
- **数据库**：MySQL
- **安全加密**：bcrypt

## 功能模块

### 用户管理

- **注册**：用户可进行注册，无身份选择
- **登录**：用户登录系统
- **个人信息修改**：用户可修改个人信息，密码加密存储

### 图书管理

- **增删改查**：实现图书基本信息的添加、删除、修改和查询
- **分类关联**：图书分类管理
- **关键词搜索**：支持按关键词搜索图书

### 借阅管理

- **图书借出/归还**：管理图书的借出与归还，自动更新状态
- **时间格式化显示**：借阅时间格式化显示

### 分类管理

- **添加与删除**：图书分类的添加和删除

### 借阅统计

- 展示总图书数、借出数、可借数、活跃借阅记录及借阅率

## 系统角色

- 系统管理员：负责图书和用户的管理
- 普通用户：进行图书借阅和归还

## 运行环境

- 操作系统：兼容Windows、Linux等
- Python版本：3.x
- 所需库：Tkinter、MySQL Connector、bcrypt等

## 部署步骤

1. 确保Python环境和MySQL数据库已正确安装
2. 安装依赖库：通过pip安装Tkinter、MySQL Connector、bcrypt等
3. 运行系统启动脚本，自动初始化数据库和表
4. 启动系统

## 目录结构

```
project/
│
├── main.py          # 系统主程序
├── database.py      # 数据库连接和操作模块
├── views.py         # 视图模块，包含界面构建
├── models.py        # 模型模块，包含数据结构定义
└── utils.py         # 辅助功能模块，如加密、验证等
```

- **安全**：用户密码使用bcrypt加密存储
- **便捷**：自动初始化数据库，简化部署流程
- **界面**：分区设计，简洁直观，易于操作
- **适用范围**：适用于小型图书馆或机构的图书管理场景
